# Demo Screenshot Instructions

Place a screenshot named `demo.png` in this directory.

## What to capture

A Claude Desktop or Claude Code conversation showing:
1. **User prompt** (e.g. "Wie hoch sind die Aktiven aller Schweizer Banken?")
2. **Tool call** visible (e.g. `snb_get_banking_balance_sheet`)
3. **Formatted response** with Markdown table showing Millionen CHF values

## Recommended dimensions

- Width: 1440px (Retina) or 720px (1x)
- Format: PNG
- File name: `demo.png`

## Example prompt ideas

- "Was ist der aktuelle EUR/CHF-Kurs und wie steht der SNB-Leitzins im Vergleich zu Fed und EZB?"
- "Zeige mir die Bilanz aller Schweizer Banken 2023 — Aktiven und Passiven."
- "Wie hat sich der Geschäftsertrag der Grossbanken in den letzten 5 Jahren entwickelt?"
